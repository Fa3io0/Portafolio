3. Diferències entre Mapa de Bits i Vectorials

1. Escalabilitat (Resolució):
Les imatges SVG (Vectorials) són fórmules matemàtiques. Si fas zoom, les línies es mantenen perfectament nítides. Mai es pixel·len.
Les imatges de Mapa de Bits (PNG/JPG) estan formades per una quadrícula de píxels. Si les amplies, es veuen borroses o "pixel·lades" (es veuen els quadrats).

2. Pes del fitxer:
Els fitxers SVG solen ocupar molt poc espai (només és text/codi), especialment per a gràfics senzills com aquests.
Els mapes de bits ocupen més espai perquè han de guardar la informació del color de cada píxel individual.

3. Edició:
En un SVG, pots canviar el color d'una forma canviant una línia de codi (ex: fill="red"). En un mapa de bits, els colors estan "fusionats" i és més difícil editar elements separats.
